# 素数与约数

## 试除法求给定的数字是否为素数

### 性质
1. 约数成对出现，若 $d$ 为 $n$ 的约数，则$\frac{n}{d}$也是其约数
2. 基于性质1，可以得到循环i的次数，即 $i \leq \frac{n}{i}$

### 基本原理
```
   由2开始遍历每个数i，若i小于n/i，则i++
    判断n是否能被i整除，若可以，则n不是素数
    否则continue

  最后若没有提前结束，则n是素数
```

### 代码

```c++
int is_su(int n)
{
    if(n < 2) return -1;
    for(int i = 2; i <= n/i; i++)
    {
        if(n%i == 0)
        {
            return -1;
        }
    }

    return 1;
}
```
